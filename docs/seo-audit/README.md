# SEO Audit Prompts — Reference Index

## Purpose

These are the authoritative SEO audit deliverables generated by IntegrePro Software LLC. They are the primary source for all page copy, schema spec, internal linking, and SEO directives. When kickoff docs (`Phase2_Kickoff_*.md`, `Phase3_Kickoff_*.md`, etc.) conflict with audit prompts in this directory, the audit prompt wins per CLAUDE.md working principle #2 (Voice rules > live-site parity > SEO audit > drafted copy).

The kickoff docs are routing and orientation documents — they summarize and sometimes paraphrase the audit outputs. When exact wording matters (title tags, H1s, opening paragraphs, FAQ answers, internal link anchor text, schema fields), read the audit prompt directly.

## Note on conversion quality

Files were converted from `.docx` to markdown using `mammoth`. Tables in the source documents render as flat cell-per-line rather than pipe-delimited markdown tables — a known mammoth limitation. All prose, verbatim page copy, em dashes, and structured lists are preserved accurately. The files are readable and correct; only table visual formatting is degraded.

## Update policy

Audit prompts are immutable historical artifacts. Do not edit. Corrections or updates come from Ryan as new prompt versions, saved under a new filename (e.g., `prompt-11-service-city-page-builder-v2.md`) with a note at the top explaining what changed and why.
